# React Redux Boilerplate

A react, redux, webpack, css modules, postcss, karma and mocha boilerplate. Complete with super simple authentication flow, tests on reducers and components. Related to [blogpost](http://pebblecode.com/blog/react-redux-unit-testing/) 

##Installation
``` git clone git@github.com:export-mike/react-redux-boilerplate.git```

```npm i ```

``` npm start ```

new shell:
``` npm run test:watch ```

### other commands:
test mocha only:
``` npm run mocha:watch ```

test karma only: 
``` npm run karma:watch ```

for CI omit :watch on your npm run command.

Build:
``` npm run build ```
